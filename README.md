# XFET(ch)
Mass download YouTube or SoundCloud songs in high quality

## Description
XFET(ch) is a Python script for mass downloading YouTube / SoundCloud audio,
by Nytelife26. It uses youtube-dl and FFmpeg to download and process songs.
